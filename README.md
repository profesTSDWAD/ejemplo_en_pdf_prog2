# ejemplo_en_pdf_prog2
>Ejemplos en Django y DRF para programación 2

Repositorio donde está el codigo de los ejemplos explicados en los encuentros sincrónicos y material de estudio. En lugar de generar un solo proyecto en la carpeta raiz del repositorio, se lo agrega en un nivel interior, para permitir tener otros proyectos independientes en el mismo repositorio, a fin de tener todos los ejemplos juntos pero ordenados.

Se utiliza como primer proyecto *"miproyecto"*, donde se harán dos aplicaciones, la primera ("saludo"), con el clasico "Hola Mundo", en la raiz de la URL de los EndPoint. Segunda aplicación ("app") se utiliza la biblioteca Django REST Framework (DRF), para visualizar otros EndPoint.
